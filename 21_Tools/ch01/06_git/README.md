
### git 工作流示意图

```shell
                                   pull
                      +-----------------------------------+
    +--------+        |         +--------+                +--------->+--------+
    |        |        |         |        |    checkout               |        |
    |        +--------+         |        +-------------------------->+        |
    |        |  fetch/clone     |        |                           |        |
    |远程仓库+----------------->+本地仓库+           +------+        |工作空间|
    |        |                  |        |           |      |        |        |
    |        +<-----------------+        +<----------+ 索引 +<-------+        |
    |        |       push       |        |  commit   |      |   add  |        |
    +--------+                  +--------+           +------+        +--------+
```
