
## 安装环境
```
    VMware version:         14.1.1
    linux kernel version:   3.10.0-862.el7.x86_64
    Linux Release:          RedHat 7.5
```

这里直接使用 pip3 方式安装。


## meson 安装

```sh
    pip3 install meson
```
缺省安装到 `/usr/local/bin/` 目录下。


## ninja 安装

```sh
    pip3 install ninja
```
缺省安装到 `/usr/local/bin/` 目录下。
