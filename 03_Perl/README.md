### 为什么要使用 perl

- 支持浮点运算
- 支持 linux 命令
- 够看的数据类型
- 虽是脚本，但速度并不慢
- 很多函数的名称和使用上对接 C 语言
- 语言级正则表达式
- 支持进程管理
- 足够丰富的模块支持，如数据库、进程间通信等
- 也可以自定义模块
- 面向对象编程
- 各类 linux 上都提供了原生支持

Perl 这种完整的语言提供了 POSIX 级别的接口，理论上在系统 API 层面上提供了一个基本上无限的扩展范围。
