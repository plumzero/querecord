从运行下面的命令开始:
```sh
    $ docker run -d -p 80:80 docker/get
    Unable to find image 'docker/get:latest' locally
    docker: Error response from daemon: pull access denied for docker/get, repository does not exist or may require 'docker login': denied: requested access to the resource is denied.
    See 'docker run --help'.
```

出问题了，切换到国内阿里云镜像，这里要修改 `/etc/docker/daemon.json` 文件，向 url 中添加 `https://alzgoonw.mirror.aliyuncs.com`:
```json
    {
        "registry-mirrors": ["https://alzgoonw.mirror.aliyuncs.com","https://dockerhub.azk8s.cn","https://reg-mirror.qiniu.com"],
        "log-driver": "json-file",
        "log-opts": {
            "max-size": "1g"
        }
    }
```

重启 docker:
```sh
    systemctl restart docker
    systemctl status docker
```
