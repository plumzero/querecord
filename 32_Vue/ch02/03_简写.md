
Vue 为 v-bind 和 v-on 这两个最常用的指令提供了特定简写。

### v-bind 简写

```html
    <!-- 完整语法 -->
    <a v-bind:href="url">...</a>

    <!-- 简写 -->
    <a :href="url">...</a>

    <!-- 动态参数的简写(2.6.0+) -->
    <a :[key]="url"></a>
```

### v-on 简写

```html
    <!-- 完整语法 -->
    <a v-on:click="doSomething">...</a>

    <!-- 简写 -->
    <a @click="doSomething">...</a>

    <!-- 动态参数的简写(2.6.0+) -->
    <a @[event]="doSomething">...</a>
```
