
POSIX 消息队列。

POSIX 消息队列 API 中的主要函数如下:
- `mq_open()`: 创建一个新消息队列或打开一个既有队列，返回一个描述符供后续进行调用。
- `mq_send()`: 向队列写入一条消息。
- `mq_receive()`: 从队列读取一条消息。
- `mq_close()`: 关闭一个消息队列。
- `mq_getattr()` 和 `mg_setattr()`: 这两个函数用来获取和修改队列的特性。每个消息队列都有一组关联的特性，其中一些特征可以在调用 mq_open() 时设置。
