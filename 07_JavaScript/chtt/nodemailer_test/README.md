### 注意

1. 这里使用授权码进行测试。测试中创建传输句柄的配置中 `auth` 字段必须填写，`auth.user` 为授权邮箱，`auth.pass` 为授权码。该授权邮箱也作为发送邮箱。

2. 发送消息体中，`from` 字段为发送邮箱，必须与授权邮箱相同。`to` 字段为接收邮箱，可以不止一个。`bcc` 字段为抄送邮箱，也可以不止一个。
