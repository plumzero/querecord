
### 删除角色报错

```sh
    postgres=# drop role airflower ;
    ERROR:  role "airflower" cannot be dropped because some objects depend on it
    DETAIL:  privileges for schema public
```

依次执行如下命令:
```sh
    postgres=# drop owned by airflower;
    DROP OWNED
    postgres=# drop role airflower ;
    DROP ROLE
```
