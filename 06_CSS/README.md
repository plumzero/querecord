
CSS(Cascading Style Sheets, 层阶样式表)是 W3C 为了弥补传统 HTML 功能之不足所开发的一种新的网页格式标准，它可以精准地设置文字大小、文字的间距，更加入了重叠图层、区块变化及绝对定位和相对定位的功能等。

通过 CSS 可以更容易进行排版，制作出更专业、更多样化的网页。
