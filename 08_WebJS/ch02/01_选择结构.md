### switch 结构

`switch` 语句可以构成多分支选择结构，其语法格式如下:
```js
    switch(表达式) {
        case 整型数值1: 语句 1;
        case 整型数值2: 语句 2;
        ......
        case 整型数值n: 语句 n;
        default: 语句 n+1;
    }
```
