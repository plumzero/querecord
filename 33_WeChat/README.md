
*善始难善终。*

[微信公众平台](https://mp.weixin.qq.com/)提供了 4 种类型的账号，分别是服务号、订阅号、小程序和企业微信。

![](img/01.png)

- 小程序是指在微信公众平台中发布小程序所使用的账号。
- 订阅号主要用于为用户传达资讯、类似报纸、杂志。
- 服务号主要用于服务交互，类似银行、114，提供查询服务。
- 企业微信主要用于公司内部通信使用，在关注企业号前要先验证身份。
